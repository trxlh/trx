# GPU 版 TRON / USDT 靓号钱包生成器

> 专门给波场 TRX / USDT 用的钱包靓号工具  
> 支持：高仿地址、连号地址、自定义前后缀、顺子号、6/7/8/9/10A 各种靓号  
> 建议搭配独立显卡或 GPU 云服务器使用，建议全程离线运行，安全感拉满。

---

## 一、项目概览

这是一个利用 GPU 并行能力暴力遍历私钥空间的 **TRON 钱包靓号生成器**。  
它可以帮助你在短时间内找到满足特定规则的地址，例如：

- USDT 收款靓号地址  
- 批量 TRON 靓号地址（方便做地址池 / 广告号）  
- 指定样式的“高仿地址”（前几位 / 后几位相似）  
- 各种连号、顺子、自定义规则的炫酷地址  

程序为单文件绿色版，无需安装繁琐依赖，可直接断网运行，全过程私钥本地生成与保存。

> ⚠️ 重要提示：本工具仅面向技术玩家与研究学习场景。  
> 请勿用于违法用途，也不要在不了解风险的情况下大额存币。  
> 所有生成的钱包私钥请自行妥善备份与加密存放。

## 三、核心特性

### 1. GPU 加速暴力枚举

- 利用多线程 + GPU（CUDA / OpenCL）的算力加速  
- 实测一张 RTX 2060 约可达到 **每秒约 1.35 亿地址尝试**（约 135 MH/s）  
- 支持单卡 / 多卡环境，适合长时间挂机跑号

### 2. 多种靓号匹配模式

- **前缀 / 后缀匹配**  
  - 例：前 5 位 `TTRX8`，后 6 位 `888888`  
- **连续字符连号**  
  - 6A、7A、8A、9A、10A 等  
- **高仿地址模式**  
  - 读取既有地址，按“前 N 位 + 后 N 位”生成高度相似地址  
- **顺子地址识别**  
  - 如 `123456`、`234567`、`ABCDE`、`CDEF01` 这类连续字符  
- **自定义规则**  
  - 支持配置简单通配 / 类正则匹配方式（具体视当前发布版本）

### 3. 自动保存与断点能力

- 运行过程中定期把命中结果写入不同的 txt 文件  
- 可按类型拆分（数字为主、字母为主、混合、顺子等）  
- 程序退出后，下次可继续运行，不影响之前成果

### 4. 导出信息完整

- TRON 地址（T 开头）  
- 对应公钥 / 哈希  
- 私钥（支持 HEX，部分版本支持 WIF 格式）  
- 有需要时，也可导出当时使用的配置参数，方便复查与验证

### 5. 多系统兼容

- **CPU 模式**：Windows / Linux / macOS  
- **GPU 模式**：只要系统识别到显卡并有 OpenCL / CUDA 支持即可  
- 适用于本地电脑、GPU 云服务器、GPU 云桌面、支持显卡透传的虚拟机

## 四、运行机制简述（TRON 地址生成）

大致流程如下（简化表述）：

1. 随机选择一个 secp256k1 曲线上的私钥 `sk`；  
2. 推导对应的公钥 `pk`；  
3. 对 `pk` 进行 `keccak256`，取结果的后 20 字节；  
4. 在前面加上波场主网前缀 `0x41`，合成 21 字节原始地址数据；  
5. 对原始地址数据进行 Base58Check 编码（双 SHA256 + 校验位）；  
6. 得到以 `T` 开头的可读 TRON 地址；  
7. 将该地址与当前设定规则（前缀、后缀、连号、高仿、顺子、自定义等）对比：  
   - 符合：写入相应结果文件  
   - 不符合：继续下一个私钥循环

整个过程重复进行，直到你手动停止或达到自己的目标数量。

## 五、文件与配置说明

### 1. 主程序

- `gd_gpu.exe`  
  - Windows 平台的主执行文件（名称以实际发布版本为准）  
  - 其他平台会对应不同后缀

### 2. `config.txt`

- 文件开头的几行数字代表“算法基数”/“随机种子”等内部参数  
- 用于微调生成分布，让高难度连号（如 9A / 10A）在合理时间出现概率更友好  
- 普通用户无需理解具体算法，只要偶尔 **改一改数字再启动** 即可换一套随机序列

### 3. `address.txt`

- 用于配置 **高仿地址** 的目标列表  
- 一行一个真实地址  
- 程序会根据你在参数中设置的“前几位 + 后几位”策略生成近似靓号  
- 例如：  
  - 只仿前 3 位 + 后 4 位  
  - 或前 4 + 后 6 等，自行调节

### 4. `diy.txt`（可选）

- 主要用于 **自定义前后 10 位样式**  
- 一行一个模板，总长度为 20 位：前 10 位 + 后 10 位  
- 举例：  
  - 如果只关心尾巴，就把前 10 位写成占位字符（如 `TTTTTTTTTT`），后 10 位写成你想要的号码  
  - 如果希望既有固定开头又有固定结尾，就在前后同时填入指定字符

### 5. 运行脚本：`点击运行.bat` / `高仿地址.bat`

- Windows 用户可通过这类 bat 脚本一键启动  
- bat 内部本质是启动命令行工具，你可以右键“编辑”后修改参数，例如：

```bat
gd_gpu.exe --lianhao 6 --down result_6A.txt
